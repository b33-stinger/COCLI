## crackmes.one CLI 0.1.0-alpha

a CLI tool for crackmes.one, login / download and search for crackmes

### Download
```bash
git clone https://github.com/b33-stinger/COCLI.git
cd COCLI
chmod +x main.py
```

### Install
pip
```
pip install requests
```
yay
```
yay -S python-requests --needed
```
